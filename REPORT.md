# Product Feedback Intelligence Dashboard (Python + Streamlit)

## 1. Project Overview
This project is a Product Feedback Intelligence Dashboard designed to help Product Managers analyze customer feedback and convert it into actionable product insights.

It performs sentiment analysis, keyword extraction, categorization of feedback, and generates roadmap recommendations.

## 2. Problem Statement
Product teams often receive large amounts of unstructured customer feedback through reviews, surveys, and support messages.
Manually reading and prioritizing this feedback is time-consuming and inconsistent.

## 3. Solution
The dashboard automates feedback analysis by:
- Cleaning and preprocessing raw feedback text
- Performing sentiment analysis (Positive/Neutral/Negative)
- Extracting top keywords to identify pain points and feature requests
- Categorizing feedback into product-related buckets
- Generating product roadmap recommendations

## 4. Key Features
- CSV Upload Support
- Sentiment Distribution Dashboard
- Top Keywords Extraction
- Feedback Category Breakdown
- Auto Roadmap Recommendations
- Downloadable CSV Report

## 5. Tech Stack
- Python
- Streamlit
- Pandas, NumPy
- NLTK (VADER Sentiment Analyzer)
- Matplotlib

## 6. Outcome / Impact
This tool helps reduce manual effort in analyzing customer feedback and supports faster, data-driven product roadmap decisions.
